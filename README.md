# Android AIDL Service Basico

Este projeto demonstra a criação de um serviço Android utilizando AIDL (Android Interface Definition Language) para comunicação entre processos (IPC) com base nos meus estudos de AOSP e arquitetura android.

O serviço expõe dados estáticos através de uma interface AIDL, permitindo que outros componentes ou aplicações consumam essas informações.

## Conceitos abordados

- AIDL (Android Interface Definition Language)
- Comunicação entre processos (IPC)
- Bound Services
- Binder
- Arquitetura cliente-servidor no Android
